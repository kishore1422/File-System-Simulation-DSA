Introduction
------------
This project simulates a hierarchical file system using a tree data structure.
Users can create, delete, navigate, edit, and view files/directories through a menu-driven console interface.

Features
	•	Create directory (mkdir)
	•	Create file (touch)
	•	Delete files & directories (rm)
	•	View file content (cat)
	•	Edit file content (edit)
	•	List items (ls)
	•	Change directory (cd)
	•	Show path (pwd)
	•	Search files/folders (search)
	•	Display full tree structure (tree)

Data Structure Used
---------------------
•	Tree - Each node represents a file or directory.

How to Run
------------
g++ -std=c++17 filesystem.cpp -o fsimulator
./fsimulator
